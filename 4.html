  • web hosting : 쉬움. 배울 건 적음.

http://127.0.0.1:5500/index.html (= localhost/index.html)
- 이것도 웹서버 켜야함.
http : Hyper Text Transfer Protocol
127.0.0.1 : Internet Protocol Address, 내 컴퓨터 자신을 가리키는 주소.
5500 : Port. 하나의 컴퓨터에 여러개의 서버가 동작할 때, 이를 구분하기 위함.
